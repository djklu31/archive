Processing Tabs for our Application

Saving all of the Tab Data:

Now that we have saved all of our url pages that contain tab data, we will go to each of those url pages to
pull tab information from. We will achieve this by loading every page into an xml file like we did earlier,
keep track of the total number of tabs on each page (different for each page for some odd reason) and sum them
(we will use this number later on to print out all the tabs using this number as individual indices for an array),
and store the values that we want for each tab value on each page by using the (int)$xml['count']; call for each
page (whew that was a long sentence). We will store all of the wanted of each tab in a huge array that will be
parsed to output as HTML data later on. In our example, this is achieved by storing all the data in an array
called $storedata[$y] and setting it equal to
"<div> Artist: $artist</div><div> Song: $song</div><div> Type: $type</div><div>
Version: $version</div><div> Tab Rating: $tabrating</div>";.
In addition to storing all the wanted pieces in an array, we will also want to store the individual urls for
each tab with the same indices as according to the tab pair in $storedata[$y]. This will allow us to pull
the actual tab up for the user after everything is printed. We will go over this in the next section.
