Searching for Tabs

When the Submit Button is Hit:

When the submit button is hit from the search form HTML page, the PHP should be notified that the button was set.
This can be done using an if statement with "if(isset($_REQUEST['search'])){}" (given that search is the name if
the submit button in the HTML form). Within this if statement, we set the POST values received by the PHP portion
sent from the HTML form. Because our API handles GET requests, we need to format our artist and song information
to cater to this specification. An issue that we could run into is handling spaces for the artist and song title.
Therefore, we need to replace spaces in both the artist and song name with a "+" symbol in order to properly use our API.

Handling Spaces in the Artist and Song Name:

As we mentioned, we need to handle spaces from our POST artist and song names retrieved from the HTML form page
in order for our API to properly process these search values. We do this by using the str_replace() function. As an
example from my PHP implementation, I set the $artist variable in my PHP to str_replace(" ", "+", $_POST['artist']);
This will replace any space found with a plus symbol in our POST request for artist. The same concept will be applied
to the song name in case there are any spaces in them (eg. song name: "The Great Gig in The Sky"). After handling
the POST values for the artist and the song name, we need to take in the requested "maximum number of tabs to be
shown on the screen". This will simply be stored as an integer with the code (int)$_POST['perpage'];. In my coding
example, this will be set to $perpage while artist and song are set to $artist and $song respectively.

Getting Tabs:

Next, we will get into the retrieving tabs from our API. First, we will create a string that will be concatenated to
the end of the API url in the value= portion (explained in the previous section). In my code example, this was done by
creating a $searchvalues variable that contained the artist string concatenated with a "+" symbol which is then concatenated
to the song name. In my code, this is shown as $searchvalues = $artistrequest . "+" . $song request;. As stated,
$searchvalues is then tacked on to the end of the url where we will get our tabs from. My coding example shows this
as: $url = "http://app.ultimate-guitar.com/search.php?search_type=title&page=$sitepage&iphone=1&value=$searchvalues";.
