Printing out the Tab:

In this section, we will learn how to print out the tabs for our user to choose from. We previously stored our HTML data to
be displayed in an array called $storedata and all of the urls associated with each value from $storedata in an array called
$taburlarray. Now we must print the tab searches to the screen so that the user can choose one to his liking. This will be
accomplished by echoing each value in our array $storedata but before we do that, there must be some error handling that we need
to take care of beforehand. If the total results summed up from all of the pages is less than the chosen "maximum number per page"
value specified by the user, we need to make sure there isn't any extraneous data outputted. To handle this, we write an if
statement that says if the total count is less than the maximum number of tabs per page, then the results that will be printed
will be equal to the total count and not to the per page value set by the user.
The sample code is: if($totalcount < $perpage){$resultsprint = $totalcount;}. If this is not the case, then the results to print will
be set to the per page amount and that amount of data will be outputted. After this error handling, we are free to echo our storeddata
up to our specified amount. To output all of the data, simple substitute the value summed from all the pages earlier with "results
to print" value. In my code, I chose to add a submit button that opens a new window to the associated url of the particular $storeddata index.
