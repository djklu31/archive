The Ultimate-Guitar Unofficial "API" Explained

The Ultimate-Guitar tab database can be accessed via GET
requests via the URL: http://app.ultimate-guitar.com/search.php.
Actual tablature can be obtained at http://app.ultimate-guitar.com/iphone/tab.php.

Searching for Tabs Using the Search URL

(http://app.ultimate-guitar.com/search.php):

Individual tabs can be searched for via GET requests to http://app.ultimate-guitar.com/search.php.
GET request variables are as follows:

search_type : The value of this variable should always be set to title.
The extended functionalities of this variable are unknown.

page : This variable indicates the current page of the tab database (numerical value).
Tabs are stored in a span of pages with a random count of results per page (the purpose of this random count is unknown).
To obtain all tab information for searches, all of these pages must be traversed with all tab
information being saved in the process. The maximum number of pages per search is indicated
in the XML file with the variable "pages" which is explained below.

iphone : This variable specifies how the tab searches and actual tabs will be presented.
A value of 1 will allow us to pull XML information for both the tab searches and actual tabs.
A value of anything else will bring us to the actual Ultimate-Guitar website which is not what we want.
value : This variable will allow us to request information about both an artist and the song requested.
The value of this variable are either an artist, a song, or both separated by a "+" symbol.
Note: Make sure all of the spaces for artist and song names are replaced by a "+" to ensure proper GET functionality.

As an example, if we wanted to search for the song "Time" by "Pink Floyd", we would set search_type to it's default
which is title, page to the starting value 1 (can be changed incrementally to see more tabs), iphone to 1 which will
allow us to obtain XML information value to "pink+floyd+time".
The URL will therefore become: http://app.ultimate-guitar.com/search.php?search_type=title&page=1&iphone=1&value=pink+floyd+time.


XML File Variables

The XML file that we obtain through a GET request to the search URL will feature a few key variables.
These variables are listed below:

Root Node:

count : The amount of tabs listed for that particular page.
page : The current page of tablature being viewed.
pages : The last page of tab information available.
total and total_found : The number of unique songs on the particular page.
bands_found : The number of unique bands found for the total search.

Super handy note: To search for values in the root node of the XML, use the syntax $xml['pages'] or $xml['count']
to access pages and count respectively. Again, for every variable in the root node that you want to access,
use that particular syntax. It could save you a ton of time.

Result Node:

name : The song name.
artist : The artist name.
version : The unique version of the tab.
url : The URL of the actual tablature.
rating : The rating out of 5 set by Ultimate-Guitar users for the quality of the tab.
type : Specifies the type of tablatures. This could be values such as guitar, bass, solo, indicating what the tablature entails.

Super handy note: To search extended nodes within the XML DOM structure, use the syntax $artist = $xml->result[$y]['variablerequested']
where $y is the index of the particular result node (think of it like an array where a number will represent that numbered result node down the XML tree).

Accessing Tabs Via ID (http://app.ultimate-guitar.com/iphone/tab.php):

Tabs can be accessed by specifying a numerical value in the "id" GET variable. For example, if a tab has the URL id of 1135979,
we will get the URL: http://app.ultimate-guitar.com/iphone/tab.php?id=1135979 via a GET request,
which is a song by the band "Periphery" with the song "Buttersnips".
