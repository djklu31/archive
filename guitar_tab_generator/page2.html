Creating a Search Form (HTML)

Firstly, we will create a search form which will send data that we want
to request to the PHP down the line. This area is very open to interpretation
but for documentation we will create a simple form with fields to search for
an artist, song and the maximum number of tabs to be shown on the page.
Because this documentation will hone in on focusing on the tab generation
portion of the project, we won't go through the specifics of the simple HTML
involved. What will be important down the line for our example will be the values
that will be sent to the PHP when the submit button is hit. As a result,
I'll go over my simple form code briefly.

My Sample Form Code:

<!DOCTYPE html>
<html lang="en"
>
<head>
<meta charset="utf-8">
<script type="text/javascript">
function openWin(url)
{
window.open(url, "mywin", "menubar=0, resizable=0, width=800, height=1000")
}
</script>
<title>Guitar Tab Generator</title>
</head>
<body>
<h1>Generate a Guitar Tab</h1>
<form action="tabgenerator.php" method="POST">
<fieldset>
<legend>Search for a guitar tab!</legend>
<div>
<p>Artist:
<input type="text" id="artist" name="artist">
</p>
</div>
<div>
<p>Song Name:
<input type="text" id="song" name="song">
</p>
</div>
<div>
<p>Maximum Tabs on Page:
<select name="perpage">
<option value ='50'>50</option>
<option value ='100'>100</option>
<option value ='250'>250</option>
<option value ='500'>500</option>
</select><br>
</p>
</div>
</fieldset>
<input type="submit" name="search" value="Submit">
</form>
</body>
</html>

Note:

The important parts of this code are the value of artist, song and
"maximum tabs on page"(perpage) portions to be sent to the PHP portion
(via POST request), which will be explained in the next section of this documentation.
One thing to note is the JavaScript portion in the header of our HTML.
This script is there to enable opening selected guitar tabs in a new window,
which we will cover down the line.
