Overview of this Documentation

What are Guitar Tabs?

Guitar tabs are a way for guitar players to read music through a diagram that
represents the six strings (assuming the guitar is a standard six string) of a guitar
and the frets on the particular string to be played. The six strings are represented
using a notation signified by a series of dashes: "--------------" the frets to be
played are signified by numbers. For example, a 3 on the string to the stop means
that the third fret on the highest string (thinnest string) should be played.

Where is the Guitar Tab Database?

For this application, we will use the Ultimate-Guitar database to pull tablature from.
There isn't an official API available to the public by this company but it's possible
to pull XML files from their iPhone app database and search for artists/songs accordingly.

The Procedure:

The documentation will first go over creating a simple HTML form to search for artists and
songs. We will also go through implementing a way to display the maximum number of tabs chosen
by the user. The main meat-and-potatoes of this documentation is the PHP portion. In this
portion of the documentation, we will learn how to receive searches from the tab database
(Ultimate-Guitar "API") and parse the received XML page to embed into our web application.
Lastly, we will learn how to pull up the actual tab for our guitar playing enjoyment.
